# AI凉茶机 — 中医体质分类决策树项目

基于决策树模型的中医九种体质自动分类系统。通过 10 个问诊问题和 3 个望诊特征，预测用户体质类型。

---

## 快速开始

### 安装

无需安装第三方库。将 `deploy/` 目录拷贝至目标环境即可。

```bash
python deploy/classifier.py          # 运行内置测试
python deploy/example.py             # 查看使用示例
```

### 基本用法

```python
from deploy.classifier import TCMClassifier

clf = TCMClassifier("deploy/model.json")
result = clf.predict("DCACABAACC", [1, 0, 1])  # features: [舌苔, 眼圈, 脸色]
print(result)  # → "B" (气虚质)
```

| 参数 | 格式 | 示例 |
|-----|------|------|
| `answers` | 长度 10 的字符串，仅含 A/B/C/D | `"DCACABAACC"` |
| `features` | 长度 3 的列表 [舌苔, 眼圈, 脸色] | `[1, 0, 1]` |
| 返回值 | 单个字母 A–I | `"B"` |

### 体质编码

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| 平和质 | 气虚质 | 阳虚质 | 阴虚质 | 痰湿质 | 湿热质 | 血瘀质 | 气郁质 | 特禀质 |

## 数据集

### 数据来源

基于周萍《基于中医体质理论与六大茶类的体质茶疗法成果研究》(2026) 的中医体质分类理论，参照《中医体质分类与判定》国家标准设计特征体系。

- **理论基础文档**：[doc/proof.md](doc/proof.md) — 体质分类标准、问题设计依据、茶疗推荐
- **问题与评判标准**：[doc/question.md](doc/question.md) — 十问问卷与体质判别速查表
- **需求文档**：[doc/proposal.md](doc/proposal.md) — 接口规格与技术方案

### 数据特征

| 类别 | 数量 | 说明 |
|-----|------|------|
| 问诊特征 (Q1–Q10) | 10 个 | A/B/C/D 四选项，对应症状严重程度 |
| 望诊特征 (F1–F3) | 3 个 | 脸色/舌苔/眼圈，0=正常, 1=异常 |
| 体质类别 | 9 种 | A–I，每类约 111 条，均衡分布 |
| 总样本量 | 1000 条 | 含真实噪声扰动（核心特征 5-10%，非核心 15-25%） |

### 数据划分

| 子集 | 样本数 | 比例 |
|-----|-------|------|
| 训练集 | 700 | 70% |
| 验证集 | 150 | 15% |
| 测试集 | 150 | 15% |

## 模型训练

### 训练方法

| 项目 | 说明 |
|-----|------|
| 模型类型 | sklearn CART 决策树（二叉判定树） |
| 候选参数 | 8 组（criterion: gini/entropy, depth: 3-7） |
| 选择策略 | 验证集 Macro F1 最优 |
| 较优参数 | `entropy, max_depth=6, min_samples_split=5, min_samples_leaf=2` |
| 对比模型 | 手写 ID3 多叉决策树（用于验证设计可行性） |
| 训练脚本 | [scripts/train_model.py](scripts/train_model.py) |

### 三集表现

| 数据集 | 样本数 | Accuracy | Macro F1 |
|-------|-------|----------|----------|
| 训练集 | 700 | 0.9829 | 0.9828 |
| 验证集 | 150 | 0.9600 | 0.9601 |
| 测试集 | 150 | **0.9600** | **0.9588** |

训练集→测试集 F1 差距 0.024，**过拟合程度低，泛化良好**。

## 标注验证

从 1000 条中随机抽取 100 条进行人工标注验证：

| 指标 | 结果 |
|-----|------|
| AI vs 人工 一致率 | **91.00%** |
| 目标 | ≥ 90% |
| 达标 | ✓ |
| 差异明细 | [data/label_diff.csv](data/label_diff.csv) |

> 56% 的差异来自气虚↔阳虚互混，为中医体质分类中的公认难点。详见 [doc/update/dev_report_phase1.md](doc/update/dev_report_phase1.md)。
